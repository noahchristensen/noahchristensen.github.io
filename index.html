Ongoing Projects

Anime Fighting Game

Influenced by Anime-Styled fighting games like Guilty Gear, Blazblue, and Melty Blood. In it's current state, the game has a multitude of movement options like double-jumping, air-dashing, running, and crouching. I was even able to successfully implement both an input buffer and command reader. A command reader allows for things like quarter circle motions resulting in special moves like a fireball.

The game still needs a lot more testing. I've initial implemented Hitstun and Hitstop using time dialation, but in a fighting game I need specific notions of frames rather than more or less arbitrary changes to the time dilation. I'd love to also start modeling some up to four or so unique characters designs and actually developing their movesets before I move on.

First Person Zombie Survival

I made this game with two goals in mind: to have a working online system so the game is playable with friends, and to get the game downloadable from steam. The game so far is not much different than Call of Duty - Zombies, with the twist being instead of guns the player must use medieval themed weapons. I've shaped out both the primary game mechanics and a preliminary online system built around those mechanics using Unreal Engine's networking. I've also created lobbies that work through Steam.

Now, I have testing to make sure the online replication is working properly. Once I know the online works around the base of the game, I'll make sure to add mechanics to polish the game feel. Lastly, The picture shown is of a map I've adapted from Unreal Marketplace to work for the game.
